Project Title: Exploratory Data Analysis (EDA)& ML of an Airbnb Dataset Live Link: https://finalprojectairbnbepsilon-5bp39abpuvl6i36sphxpej.streamlit.app

Description: This interactive Streamlit app presents a comprehensive exploratory data analysis of an Airbnb listings dataset. It enables users to:

Explore key metrics such as listing counts, price distributions, room types and review statistics.

Visualize geographic distributions of listings (by location, neighborhood, etc.).

Investigate how various listing features relate to price and availability.

compare subsets of the data (by room type, neighbourhood, price range) .

Gain actionable insights for hosts, property managers or analysts interested in short-term rental trends.
# Airbnb Price Predictor

Predicts nightly prices for Airbnb listings in the USA using a trained XGBoost pipeline.

## Features
- Predict price based on property, location, and amenities.
- Interactive maps with Plotly and Streamlit-Folium.
- Handles missing values and categorical encoding automatically.
